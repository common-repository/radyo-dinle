== Description ==

Canlı Radyo Dinle eklentisini widget kullanılarak wordpress sitenize ekleyebilirsiniz.

- Ekletimizi yükleyip aktif ettikten sonra bileşenler kısmında bölüm görürsünüz bileşeni İsteğiniz yere ekleyebilir taşıyabilirsiniz.

== Installation ==

İndirdiğiniz Radyo Dinle WordPress Eklentisini Zipli Dosyadan Çıkarıp /wp-content/plugins/` klasörüne yükleyiniz. Eklentiyi Aktif Edip Bileşenler bölümünden Görünmesini İstediğiniz Yere Sürükleyip Bırakıyorsunuz...
